## 一、产品概述

### 1.1 产品名称
**珠江小智 AI校园助手**

### 1.2 产品定位
面向华南农业大学珠江学院大一新生的智能问答助手，以「AI学长」身份提供校园信息咨询，帮助新生快速了解学校各项事务。

### 1.3 产品愿景
让每一位华珠新生都有一个7×24小时在线的「AI学长」，随时解答校园相关问题。

### 1.4 目标用户
- **主要用户**：华珠大一新生（约5000人/年）
- **次要用户**：在校生、考生家长、学校管理人员

## 三、功能需求

### 3.1 首页（P0 核心）

#### 功能点
- 品牌展示：标题「珠江小智」+ 副标题「AI校园助手」
- 欢迎语：「你好，我是你的华珠AI学长，有什么校园问题都可以问我。」
- 5个快捷入口图标卡片：

| 图标 | 标签 | 点击行为 |
|------|------|----------|
| 🎓 | 教务咨询 | 跳转聊天页，预填提示词「请帮我了解教务相关信息」 |
| 🏠 | 宿舍生活 | 跳转聊天页，预填提示词「请帮我了解宿舍相关信息」 |
| 📚 | 新生指南 | 跳转聊天页，预填提示词「请帮我了解新生报到信息」 |
| 🍜 | 校园生活 | 跳转聊天页，预填提示词「请帮我了解校园生活信息」 |
| 🎯 | 成长规划 | 跳转聊天页，预填提示词「请帮我了解学业规划和成长建议」 |

- 输入框：首页可直接输入问题，回车或点击发送进入聊天页
- 底部推荐问题（3-5个热门问题）

#### 交互说明
- 快捷入口点击有轻微动画效果
- 输入框获得焦点后有发光边框
- 页面加载有渐入动画

---

### 3.2 AI聊天页面（P0 核心）

#### 功能点

**对话区域**
- 类ChatGPT的对话界面
- 用户消息右对齐（蓝色气泡）
- AI回答左对齐（灰色气泡），支持Markdown渲染
- AI回答时显示「正在输入...」动画（三个跳动的点）
- 自动滚动到最新消息
- AI身份头像：「学长」头像

**输入区域**
- 底部固定输入栏
- 支持多行输入（TextArea自动调整高度，上限4行）
- 发送按钮（或Enter发送，Shift+Enter换行）
- 输入框placeholder：「输入你的问题，比如：怎么选课？」

**侧边栏**
- 新建对话按钮
- 历史对话列表（按时间倒序）
- 每个对话显示标题+时间
- 点击历史对话可切换
- 删除对话按钮（悬停显示）

**推荐问题**
- 对话为空时显示推荐问题卡片（可点击发送）
- 基于当前对话主题推荐相关问题

**反馈功能**
- 每条AI回答下方显示 👍👎 按钮
- 点赞/踩后显示确认状态
- 点踩后可选择原因（不准确/不完整/无关/其他）

**页面信息**
- 顶部显示当前对话标题
- 对话标题自动生成（基于用户第一个问题）

#### 交互说明
- 发送消息后输入框清空
- 新对话默认标题「新对话」
- 第一个问题后自动生成标题

### 3.3 新生服务中心（P0 核心）

#### 功能点

**Yu学长信息展示**
- 头像/名称：「Yu学长」
- 身份说明：「华南农业大学珠江学院新生服务助手」
- 微信号展示：**yux0620x**（可复制）
- 服务范围标签：
  - 📋 新生问题咨询
  - 🏠 校园生活指导
  - 🛍️ 新生用品咨询
  - 🚗 学车咨询
  - 🎯 校园服务推荐

**微信二维码**
- 「添加Yu学长微信」按钮
- 点击后弹出模态框，展示微信二维码图片
- 支持长按保存（移动端）

**展示位置（4处）**

| 位置 | 展示形式 |
|------|----------|
| 首页底部 | 「新生服务中心」卡片，包含Yu学长头像+简介+微信按钮 |
| AI聊天页面底部 | 固定底栏提示文字「需要人工帮助？联系 Yu学长」+ 微信按钮 |
| 新生服务详情页 | 完整信息展示：Yu学长全部服务介绍+二维码 |
| 关于页面 | Yu学长介绍 + 联系入口 |

**AI自动推荐规则**
- ❌ 不在常规AI回答中主动推广微信
- ✅ 仅在以下场景推荐联系Yu学长：
  1. 用户主动询问「哪里买」「怎么买」「购买渠道」等新生用品相关
  2. 用户明确表示需要「人工帮助」「人工客服」「真人」
  3. 用户询问「学车」「驾校」「驾照」相关问题
  4. 用户对AI回答不满意（连续追问3次+）

### 3.4 知识库管理后台（P1 重要）

#### 功能点

**文档管理**
- 文档列表（表格视图）
  - 列：文件名、类型、大小、上传时间、状态
  - 支持搜索、排序、分页
- 上传文档按钮
  - 支持 PDF、Word (.docx)、Markdown、TXT
  - 拖拽上传区域
  - 批量上传支持
  - 上传后自动解析并存入向量数据库
- 删除文档（确认弹窗）
- 文档处理状态显示
  - 上传中 / 解析中 / 已入库 / 失败

**网页导入**
- 输入URL，自动抓取内容并解析入库
- 显示抓取状态

**知识库统计**
- 文档总数
- 总字符数
- 向量分段数
- 最后更新时间

## 四、非功能需求

### 4.1 性能
- 首页首次加载 ≤ 3秒
- AI回答首字延迟 ≤ 2秒
- API响应时间 ≤ 500ms（非AI接口）

### 4.2 安全
- 管理后台需登录认证
- API接口限流（单个IP每分钟最多20次请求）
- 输入内容过滤（防止注入攻击）
- XSS防护

### 4.3 可用性
- 移动端友好（触摸操作、自适应布局）
- 键盘快捷键支持（Enter发送、Shift+Enter换行）
- 错误提示友好（非技术语言）

### 4.4 可维护性
- 前后端分离架构
- 知识库内容与代码分离
- 模块化设计，便于扩展

## 六、AI回答规范

### 6.1 身份设定
> 你是一名熟悉华南农业大学珠江学院校园生活的优秀学长，你的名字叫"珠江小智"。

### 6.2 回答原则
1. **知识库优先**：优先使用学校知识库内容回答
2. **诚实原则**：不确定的信息必须说明「这个我不太确定，建议你咨询辅导员或查看学校官网」
3. **不编造**：绝不伪造学校政策、规定
4. **亲切风格**：语气亲切简单，像学长和学弟学妹聊天
5. **新生友好**：使用通俗易懂的语言，避免过多专业术语

### 6.3 回答示例

**用户**：「怎么选课？」

**AI回答**：
> 你好！新生选课一般需要登录学校教务系统，在规定时间内进行。体育课等热门课程比较抢手，建议提前准备好备选方案哦～
>
> 选修课有线上和线下两种：线上选修课是看视频做测验，线下选修课要到教室上课（期末通常是开卷考试）。
>
> 如果你告诉我你的专业，我可以进一步帮你分析需要选哪些类型的课程～

### 6.4 安全规则
- 不回答政治敏感问题
- 不提供医疗建议（引导去医务室）
- 不提供法律建议
- 不参与角色扮演之外的对话

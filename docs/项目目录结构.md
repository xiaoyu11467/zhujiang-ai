## 二、开发路径说明

| 路径 | 说明 |
|------|------|
| `frontend/` | React前端，开发端口 **5173** |
| `backend/` | NestJS后端，开发端口 **3000** |

## 三、环境变量

```env
# .env.example

# ---------- 数据库 ----------
DATABASE_URL=postgresql://postgres:password@localhost:5432/zhujiang_ai

# ---------- ChromaDB ----------
CHROMA_URL=http://localhost:8000
CHROMA_COLLECTION=campus_knowledge

# ---------- LLM ----------
# DeepSeek API（主要）
DEEPSEEK_API_KEY=sk-your-deepseek-key
DEEPSEEK_BASE_URL=https://api.deepseek.com/v1
DEEPSEEK_MODEL=deepseek-chat

# OpenAI API（备用/Embedding）
OPENAI_API_KEY=sk-your-openai-key
OPENAI_BASE_URL=https://api.openai.com/v1
OPENAI_MODEL=gpt-4o-mini

# Embedding模型
EMBEDDING_MODEL=text-embedding-3-small
# 或使用本地模型：bge-large-zh-v1.5

# ---------- 认证 ----------
JWT_SECRET=your-jwt-secret-key
JWT_EXPIRES_IN=7d

# ---------- 应用 ----------
PORT=3000
NODE_ENV=development
CORS_ORIGIN=http://localhost:5173
```

## 四、启动顺序

```
1. docker-compose up -d postgres chromadb    # 启动基础设施
2. cd backend && npm run start:dev           # 启动后端 (3000)
3. cd frontend && npm run dev                # 启动前端 (5173)
4. npm run seed:knowledge                    # 初始化知识库
5. npm run create:admin                      # 创建管理员
```

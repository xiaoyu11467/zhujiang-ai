## 使用说明

本知识库专为「珠江小智 AI校园助手」RAG系统设计，所有内容均基于公开资料整理。

**RAG检索策略：**
1. 用户问题先经过意图分类（报到/选课/宿舍/奖学金/转专业/校园设施 等）
2. 在对应分类目录下进行语义搜索
3. 返回最相关的知识片段
4. 由LLM结合检索结果生成回答

**注意：**
- 学校政策可能随时更新，请定期维护知识库
- 标有「待确认」的内容需要通过学校官方渠道进一步核实
- AI回答时必须遵循「不确定的信息必须说明不确定」原则

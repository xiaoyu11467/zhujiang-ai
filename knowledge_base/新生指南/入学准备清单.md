# 新生入学准备清单

> 来源：学校官方通知、学长学姐经验整理
> 更新时间：2025年8月

---

## 一、证件与材料

- [ ] 录取通知书原件
- [ ] 高考准考证原件
- [ ] 身份证及正反面复印件（建议5-10份）
- [ ] 户口本复印件（建议5-10份）
- [ ] 学籍档案（密封完整，切勿私自拆封）
- [ ] 一寸/小一寸免冠照片（红、白、蓝底各10张，保留电子版）
- [ ] 党团组织关系转接材料
- [ ] 户口迁移证（仅需迁移户口的同学）
- [ ] 贫困证明材料（如需申请助学资助）
- [ ] 少量现金备用

---

## 二、床上用品（床尺寸：90cm × 190cm）

- [ ] 床垫（建议厚度5-8cm）
- [ ] 床单、被套、枕套（建议2套换洗）
- [ ] 棉被/空调被（广东夏天炎热，建议薄被为主）
- [ ] 枕头
- [ ] 蚊帐/一体式床帘（建议规格：宽90×长190×高110cm）
- [ ] 凉席

---

## 三、生活用品

- [ ] 洗漱用品（牙刷、牙膏、毛巾、沐浴露、洗发水等）
- [ ] 拖鞋、日常鞋（建议多备一双运动鞋）
- [ ] 衣架、晾衣杆
- [ ] 洗衣液/洗衣粉
- [ ] 水杯/水壶（建议大容量1L以上，军训必备）
- [ ] 餐具（饭盒、筷子、勺子）
- [ ] 雨伞/雨衣（广东多雨）
- [ ] 小型医药箱（感冒药、创可贴、清凉油、藿香正气水等）
- [ ] 充电宝、数据线
- [ ] 排插/插线板
- [ ] 小锁（用于锁柜子）

---

## 四、学习用品

- [ ] 笔记本电脑及配件
- [ ] 笔、笔记本
- [ ] U盘
- [ ] 台灯/床头灯
- [ ] 书包

---

## 五、军训特殊准备

- [ ] 防晒霜（SPF50+，PA++++）
- [ ] 芦荟胶（晒后修复）
- [ ] 驱蚊水、风油精
- [ ] 藿香正气水（防中暑）
- [ ] 润喉糖
- [ ] 葡萄糖（防低血糖）
- [ ] 柔软厚鞋垫（军训鞋底较硬）
- [ ] 纯棉T恤（内搭吸汗）
- [ ] 皮带（军训裤腰可能偏大）
- [ ] 湿巾/纸巾
- [ ] 大容量水杯（1L以上）

---

## 六、建议到校后再买的

- 置物架、收纳箱（需实地测量尺寸）
- 椅子/靠垫
- 扫把、拖把等清洁工具（可与室友合买）
- 桶装水/饮水机相关（宿舍已有饮水机）

---

## 七、不建议携带的

- 大功率电器（被查到会被没收）
- 过多现金（校园消费以手机支付为主）
- 过多衣物（宿舍空间有限，可换季时让家人寄送）

---

## 八、提前准备的建议

1. **行李多怎么办？** → 外省同学可将大件行李提前快递至学校驿站
2. **家长送学住哪里？** → 建议提前预订学校周边酒店，开学季房源紧张
3. **需要办银行卡吗？** → 如已有银行卡则不需要，奖学金和退费需用到；没有的话开学后银行会到校办理
4. **教材怎么办？** → 每年一次性交800元教材费，大一上学期统一发教材

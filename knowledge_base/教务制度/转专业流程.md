# 转专业流程

> 来源：教务处《转专业工作实施细则》（2024版）
> 更新时间：2025年3月

---

## 一、基本政策

华南农业大学珠江学院对**转入转出名额不设限制**，文理科可以互转，不受高考选考科目限制。

> 整体来说转专业相对容易，但热门专业竞争可能较大。

---

## 二、申请条件

### ✅ 可以申请

1. 通过高考录取的在籍全日制学生，**一至三年级在校期间各有一次机会**
2. 专升本录取学生，一年级有一次转入其他专升本专业的机会
3. 学习态度端正，遵守校纪校规，无违纪作弊记录
4. 身体条件符合拟转入专业招生的要求
5. 对拟转入专业有一定兴趣和专长

### 🚫 不能申请

1. 以特殊招生形式录取的学生（国家有相关规定或录取前与学校有明确约定的）
2. **艺术类、体育类学生申请转入普通类专业**（艺术类只能转艺术类，普通类只能转普通类）
3. 在发放录取通知书和新生入学报到环节
4. 休学或保留学籍期间的学生
5. 其他有失公平公正的情况

---

## 三、转专业时间流程（每学年第一学期）

| 时间节点 | 事项 |
|----------|------|
| 第8周 | 各二级学院制定转专业实施方案 |
| 第9周 | 教务处发布通知，公布各学院方案 |
| 第10周前 | 学生提交申请材料 |
| 第10-11周 | 转出学院初审，材料转至拟转入学院 |
| 第11-12周 | 转入学院组织考核（笔试+面试），择优录取 |
| 第12-13周 | 转入学院走OA流程审批 |
| 第13-14周 | 校长办公会议审核，公示7日 |
| 次学期 | 凭学校文件到转入专业报到注册，插班跟读 |

---

## 四、提交材料

- 《转专业审批表》
- 手写转专业申请书
- 《课程认定表》

---

## 五、重要注意事项

1. **仅有一次机会**：一经办理转专业手续，不得转回原专业，务必慎重！
2. **大三学生需降级**：三年级申请转专业，须降至二年级学习
3. **学分处理**：
   - 已修且符合转入专业要求的课程 → 可免修
   - 不符合转入专业要求的课程 → 可作为公共选修课记入学分
   - 转入专业要求但未修读的课程 → 必须补修
4. **学费标准**：转专业后按转入专业标准缴纳学费
5. **毕业审核**：按转入专业的要求审核毕业资格和学位授予资格
6. **考核方式**：笔试+面试，择优录取

---

## 六、建议

- 申请前多旁听目标专业课程，确认自己对新专业有真实兴趣
- 与学长学姐和辅导员充分沟通
- 做好职业规划，了解目标专业的就业方向和发展前景

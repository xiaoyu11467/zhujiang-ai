# 学分要求

> 来源：教务处《学分认定和转换工作实施办法》、培养方案
> 更新时间：2025年5月

---

## 一、毕业学分要求

- 学生在规定学习年限内，修完教学计划规定的全部内容
- 成绩合格，达到毕业要求的，准予毕业，颁发毕业证书
- 符合学士学位授予条件的，授予学士学位

> 具体学分总数因专业而异，请查看本专业培养方案。

---

## 二、学分类型

1. **必修课学分**：按培养方案必须修读
2. **选修课学分**：分为线上和线下选修，需修满规定数量
3. **实践教学学分**：包括实习、实训、毕业论文/设计等

---

## 三、学分认定与转换

依据《华南农业大学珠江学院学分认定和转换工作实施办法（试行）》（2025年5月），以下成果可认定学分：

| 类型 | 可认定成果 |
|------|-----------|
| 资格证书 | 国家职业资格证书、职业技能等级证书 |
| 竞赛获奖 | 各级学科竞赛获奖证书 |
| 在线课程 | 在线开放课程学习证明 |
| 培训经历 | 非学历教育培训证书 |
| 自学考试 | 高等教育自学考试课程合格证 |
| 职业经历 | 相关工作经历证明 |
| 创新创业 | 创业实践、专利版权等 |
| 志愿服务 | 社会实践、志愿服务证明 |

### 成绩换算规则
- 百分制：按原成绩录入
- 等级制（A、B、C…）：对应百分制转换标准
- 五级制（优秀/良好/中等/及格/不及格）：对应百分制转换标准
- 无具体成绩的业绩类：按**75分**或**中等**登载

### 办理流程
1. 每学年第一学期第一周填写《学生学分认定和转换申请表》
2. 附上证明材料
3. 提交至所在二级学院审核

---

## 四、重修与学分

- 除任意选修课外，所有不及格课程均应重修
- 在学习年限内**重修次数不限**，直至取得学分
- 重修成绩按实际考试成绩记载，并计算绩点
- 通过补考、重修获得的成绩，在成绩册中均**予以标注**

---

## 五、学业警示

按不及格课程学分数，给予：
- 学业警示
- 留级/降级
- 直至**退学**处理

> 务必认真对待每门课程，避免因挂科影响学业进度。

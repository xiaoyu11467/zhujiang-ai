# 图书馆

> 来源：学校官方、学长学姐经验
> 更新时间：2025年

---

## 一、从化校区图书馆

### 基本信息
| 项目 | 详情 |
|------|------|
| 位置 | 20栋C区 |
| 建筑面积 | 1.83万平方米 |
| 藏书量 | 120余万册 |
| 报刊 | 中外文报刊500余种 |

### 楼层分布
| 楼层 | 功能区 |
|------|--------|
| 1楼 | 入口 |
| 2-5楼 | 各类图书区 |
| 6楼 | 报刊室、**自习室**、**考研室**（有空调） |

### 空调情况
- 2-5楼：**无空调**
- 6楼：**有空调**（考研和自习备考首选）

### 开放时间
| 日期 | 时间 |
|------|------|
| 周一至周四 | 8:00 — 22:00 |
| 周五 | 8:00 — 17:30 |
| 周六至周日 | 8:00 — 22:00 |

### 设施
- 自习座位配有充电口
- WiFi全覆盖
- 7栋部分教室也可作自习室使用

### 入馆方式
- 刷校园卡或学生证入馆
- 借书需办理借阅手续

## 二、四会校区图书馆

- 新建场馆，玻璃砖墙设计
- 分类清晰的书库
- 温馨灯带、科学设计的桌椅
- 自习座位充足，环境安静舒适
- 现代化设施完善

---

## 三、图书馆使用小贴士

1. 期末/考研季座位紧张，建议早点去，尤其是6楼空调区
2. 借书需按时归还，逾期可能有罚款
3. 保持安静，手机调至静音模式
4. 贵重物品随身携带，不要用书占座
5. 周五下午闭馆较早，注意安排时间

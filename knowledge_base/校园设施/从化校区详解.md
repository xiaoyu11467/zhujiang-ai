## 二、2022年新扩建

从化校区新扩建约**13.5万平方米**，新增：
- 新教学楼
- 创新创业中心
- 体育馆
- 礼堂
- 教师公寓
- 学生公寓
- 人才公寓

扩建后可容纳**22000名学生**。

## 三、教学设施

### 实验室与实训
- 各类实验实训室共**119个**
- 省级实验教学示范中心**5个**
- 省级智能家居产业学院**1个**
- 省级大学生校外实践教学基地**10个**
- 校外实习实训基地**110多个**

### 特色设施
- **融媒演播厅** — 专业级媒体制作
- **影视摄影棚** — 影视专业实训
- **多功能报告厅** — 学术活动
- **智慧教室** — 互动教学
- **影视工作坊** — 影视创作
- **大商科综合实训室** — 经管类实训
- **珠江剧院** — 文艺演出
- **学术交流中心** — 学术会议
- **校史馆** — 学校历史
- **学生事务中心** — 综合服务（21栋楼下）

## 四、院系设置（从化校区特色）

从化校区突出**传媒艺术专业特色**，主要学院：

| 学院 | 特色 |
|------|------|
| 大湾区影视学院 | 戏剧影视导演、摄影、影视摄影与制作 |
| 国际传播学院 | 广播电视编导、传播学、网络与新媒体 |
| 艺术设计学院 | 视觉传达设计、环境设计（省一流）、数字媒体艺术 |
| 体育学院 | 休闲体育（2025年首招） |
| 博雅国际书院 | 低空物流等特色方向 |
| 马克思主义学院 | 思政教育 |
| 继续教育学院 | 成人教育 |
| 国际教育学院 | 海外升硕培养计划 |

## 六、食堂与商业

### 食堂
- **三个食堂**（第二至第四饭堂）
- 分别位于11栋、14栋、18栋
- 第四饭堂最大、种类最丰富
- 香港街小吃摊位
- 人均约13-15元

### 品牌入驻
- **瑞幸咖啡**
- **蜜雪冰城**
- 益禾堂（奶茶）
- 各类小吃品牌

### 其他商业
- 19栋一楼：超市、打印店、奶茶店、水果店、面包店
- 20栋C一楼（图书馆）
- ATM机

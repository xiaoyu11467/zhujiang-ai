## 五、宿舍通用配置

| 项目 | 详情 |
|------|------|
| 电梯 | ✅ **所有宿舍楼都有电梯** |
| 床尺寸 | 190cm × 90cm |
| 床帘建议 | 宽90×长190×高110cm |
| 书桌 | L型书桌 + 衣柜 |
| 热水 | 17:30-19:00 + 21:00-23:00（部分楼栋24小时） |
| 门禁 | 周日-周四 23:00，周五周六 24:00 |
| 水电费 | 自付，电约0.65元/度，人均月均50元（夏天空调更高） |
| 智能门锁 | 宿舍卡或蓝牙（小程序"51密码电子锁"） |
| 网选宿舍 | 线上自选，先缴费先选 |

---

## 六、选宿舍建议

1. ✅ **先缴费先选**，收到录取通知后尽快缴费
2. ✅ 提前了解各楼栋图，列出3-5个优先选择
3. ✅ 15栋/21栋有马桶，22栋四人间为主，26/27栋高标最贵
4. ✅ 14栋/18栋楼下就是食堂，方便但可能较吵
5. ⚠️ 学校表示不强制选择高标，但往年有过争议，选之前看清楚

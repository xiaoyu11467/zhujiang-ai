# 快递与外卖指南

> 来源：学校官方公众号、小红书学生分享
> 更新时间：2025年

---

## 从化校区

### 📮 快递地址
```
广东省广州市从化区江埔街道锦洞路168号华南农业大学珠江学院
邮编：510900
```

### 📦 三大驿站

| 驿站 | 位置 | 可取的快递 | 营业时间 | 取件方式 |
|------|------|-----------|----------|----------|
| **菜鸟驿站** | 7栋教学楼旁 | 天猫/邮政/韵达/极兔/申通/中通/京东/德邦/丹鸟 | 9:00-19:30 | 取件码 + 菜鸟App/淘宝扫码 |
| **妈妈驿站** | 四饭旁（18栋旁） | 圆通/EMS | 9:00-19:30 | 不需要身份码，直接取 |
| **顺丰驿站** | 四饭旁（18栋旁） | 顺丰 | 9:00-19:00 | 微信扫码，可寄件/上门取件 |

### 🛵 外卖

- 外卖送到**校门口**自取
- 校内跑腿代拿：小件2元起，送宿舍楼下
- 通过美团/饿了么下单
- 校内有麦当劳、塔斯汀、瑞幸、库迪、古茗等，外卖不一定需要

## 开学寄行李建议

1. 外省同学可提前3-5天寄出大件行李
2. 开学当天驿站人多，建议错峰取件
3. 贵重物品随身携带，不要寄
4. 标注好姓名+电话，方便查找
